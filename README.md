# Gestor_de_Comanda (WIP)
### Work in Progress
Proyecto de desarrollo de una app para la gestión de la comanda de un restaurante.

## Objetivo
Estre proyecto resuelve una problematica real. Administras las comandas del negocio de mi familia "El marsiquero de Acapulco". Un restaurante de mariscos que maneja una carta tipo Buffet.

El objetibo es crear una pantalla "Dashboard" en donde se muestren ordenados de mayor a menor urgencia y con un código de colores (rojo, amarillo, verde) el listado de platillos pendientes.
